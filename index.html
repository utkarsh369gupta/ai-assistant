About the Project

The Jarvis Desktop Assistant is a voice-controlled virtual assistant that can automate tasks, manage apps, and provide real-time information like IPL scores and weather updates. Designed to be versatile, it simplifies common tasks through natural language commands.

Features

  • Open and close applications
  • Browser tab management
  • Voice commands for task automation
  • Real-time data retrieval (weather, IPL scores)
  • Focus mode to minimize distractions
  • Password protection for key features

Setup Instructions

  1. Clone this repository:
    2. git clone https://github.com/utkarsh369gupta/ai-assistant.git
  2. Install the required dependencies:
    3. pip install -r requirements.txt
  3. Run the main.py file:
    4. python main.py
